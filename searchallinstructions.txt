============================================================
        ^^^^ TOOL CRIB DATABASE REFERENCE ^^^^
============================================================

The 'SEARCH ALL' mode is a live, stackable filtering system. 
When you enter this mode, you see the entire inventory 
immediately. Use the following tools to find your items.

- NO QUOTES: Treats your input as one exact phrase.
  * Example: [S] HELICAL SOLUTIONS 
    (Looks for that exact brand name)
- WITH QUOTES: Activates "OR" logic between terms.
  * Example: [S] "END MILL" "DRILL"
    (Shows all End Mills OR all Drills)

REFINING (THE "AND" OPERATION)
---------------------------------
To narrow results down, apply filters one at a time:
  1. [S] "END MILL" "DRILL"  -> (Shows both categories)
  2. [S] .250                -> (Narrows that list to only 1/4")
  3. [S] HARVEY              -> (Narrows further to only Harvey)

NEGATIVE SEARCH (EXCLUDE)
----------------------------
Use the [E] key to filter out the "noise."
- Example: If your search for Drills shows too many Spot Drills, 
  press [E] and type: SPOT
- Example: Press [E] and type: "MOVED" 
  (To hide outdated or tagged inventory)

TIPS:
- WATCH THE PATH: The 'SEARCH PATH' at the top of the screen 
  tracks every filter applied (e.g., SEARCH ALL > DRILL > NOT SPOT).
- STACK FILTERS: You can [S] Include a brand, then [E] Exclude 
  a size, then [S] Include a flute number to find a specific tool.
